Introduction

A revised version of the NBN Gateway web services (NBN Gateway API) will be released later in 2013. Ahead of that release it is recognised that some consumers may wish to begin to develop (or redevelop) applications and also contribute to more generally testing the new suite of services. With this in mind a BETA version of the API is being made available as a limited release. While various steps have been taken to minimise the inadvertent leakage of data through this period, the fact that the system is still under development means there remains a risk of such leakage occurring. The primary reason for these restrictions is to manage this possibility.

This document is a license for that BETA version of the NBN Gateway API. It contains a number of restrictions and conditions that apply to the use of the API and its documentation. The conditions will be relaxed when the API enters into full release.
