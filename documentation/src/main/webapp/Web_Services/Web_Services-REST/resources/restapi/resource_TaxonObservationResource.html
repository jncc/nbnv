/taxonObservations

Mount Point: /api/taxonObservations

GET

Returns a list of Taxon Observations matching the given serach parameters

Parameters

name description type default
user The current user, determines what datasets they have access to (Injected Token no need to pass) tokenuser
startYear The start year of the desired range query -1
endYear The end year of the desired range query -1
datasetKey Datasets to search in query
ptvk Taxon Version Keys to search for query
spatialRelationship Any spatial relationship information required query overlap
featureID Any required feature ID query
sensitive If the results should include sensitive records or not query 1
designation Any required designations query
taxonOutputGroup Any required taxon output groups query
gridRef Any grid references to search within query
polygon WKT WGS-84 polygon filter query
absence Whether the results should be limited to just absence records (true) or just presence records (false). If this parameter is missing then both absence and presence records are returned. Valid values are 'true' and 'false' query

Response Body

element: taxonObservation
media types: application/json

A list of Taxon Observations conforming to the provided search parameters

Status Codes

code description
200 Successfully returned a list of taxon observations which match the given filter
