/organisations/{id}/metadata

Mount Point: /api/organisations/{id}/metadata

GET

Return a specified organisation from the core database, if the user is a administrator of the specified organisation or return a 403 Forbidden error

Parameters

name description type default
user The current user (Must be organisation admin of this organisation) (Injected Token no need to pass) tokenorganisationuser
id An organisation ID path

Response Body

element: organisation
media types: application/json

A specified organisation from the core database

Status Codes

code description
200 Successfully returned the selected organisation
403 The current user is not and admin of the selected organisation

POST

Update the details of an organisation, if the user is an admin of the specified organisation, otherwise return a 403 Forbidden error

Parameters

name description type default
user The current user (Must be an admin of this organisation) (Injected Token no need to pass) tokenorganisationuser
id The id of an organisation path
name The name of the organisation form
abbreviation The abbreviation of the organisation form
summary A summary about the organisation form
address An address for the organisation form
postcode A postcode for the organisation form
phone A contact phone number for the organisation form
website The organisations website address form
contactName A contact for this organisation for the NBN form
contactEmail A contact email for this organisation for the NBN form

Response Body

element: opResult
media types: application/json

An OpResult detailing the success or failure of this operation

Status Codes

code description
200 Successfully completed operation, may have failed check the OpResult for details
403 The current user is not and admin of the selected organisation
