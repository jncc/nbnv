/gridMapSquares/{ptvk }

Mount Point: /api/gridMapSquares/{ptvk }

GET

Returns a zip file containing a list of grid map squares filtered by the given parameters, specifically the Taxon Version Key supplied

Parameters

name description type default
user The current user (determines what records are available) (Injected Token no need to pass) tokenuser
ptvk The Taxon Version Key we are looking for path
resolution What resolution we are looking for query
band A list of bands query
datasets A list of datasets to restrict the search to query
feature An identifier for a Vice County query

Response Body

element: (custom)
media types: application/x-zip-compressed

A zip containing a list of grid squares filtered by the given parameters
