FeatureResource

A jersey resource which provides bounding boxes for certain features on the gateway

The following resources are applicable:

  • /features/{id}
  • /features/siteBoundaries

/features/{id}

Mount Point: /api/features/{id}

GET

Return a specified feature with a numerical ID

Parameters

name description type default
id A Numerical Feature ID path

Response Body

element: feature
media types: application/json

A specified feature with a numerical ID

Status Codes

code description
200 Successfully returned the requested feature
204 Could not find the requested feature
