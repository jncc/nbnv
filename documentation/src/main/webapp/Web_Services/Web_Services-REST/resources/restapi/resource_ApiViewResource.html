ApiViewResource

The following resources are applicable:

  • /apiViews/{datasetKey }
  • /apiViews/{datasetKey }/csv

/apiViews/{datasetKey }

Mount Point: /api/apiViews/{datasetKey }

GET

Return a list of api observation views which give the filters used and the number of records viewed at via the API, includes portal operations which return full records to the user.

Parameters

name description type default
user The current user, must be admin of the dataset (Injected token no need to pass) tokendatasetadminuser
datasetKey The dataset key of the dataset path
startDate (Optional) A start date to search for views from (must have an end date) query
endDate (Optional) An end date to search for views until (must have a start date) query

Response Body

element: apiObservationViewStatistic
media types: application/json

A list of ApiObservationViewStatistic's for this dataset and any date filters

Status Codes

code description
200 Successfully returned a list of all API observation view statisics, grouped by dataset

/apiViews/{datasetKey }/csv

Mount Point: /api/apiViews/{datasetKey }/csv

GET

Return a zip file containing a CSV list of api observation views which give the filters used and the number of records viewed at via the API, includes portal operations which return full records to the user.

Parameters

name description type default
user The current user, must be admin of the dataset (Injected token no need to pass) tokendatasetadminuser
datasetKey The dataset key of the dataset path
startDate (Optional) A start date to search for views from (must have an end date) query
endDate (Optional) An end date to search for views until (must have a start date) query

Response Body

element: (custom)
media types: application/x-zip-compressed

A ZIP file containing a CSV list of ApiObservationViewStatistic's for this dataset and any date filters
