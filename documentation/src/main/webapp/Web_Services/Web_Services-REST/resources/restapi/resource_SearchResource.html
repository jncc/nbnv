/search

Mount Point: /api/search

GET

Unrestricted search, searches for any reference to a given search term in all searchable objects in the database

Parameters

name description type default
rows The number of rows to show in each page of search results query 25
start The page that the user wants to start displaying the results at query 0
sort If we should sort the results or not query
order The order in which we should sort the results query asc
q The search term query
exclude a record type to exclude from the search results (eg the IMT requires the exclusion of records that are of type 'organisation') query

Response Body

element: (custom)
media types: application/json

Search Results for this search term

Status Codes

code description
200 Successfully returned SOLR results across all searchable objects for a given term
