downloadStat element

Type: downloadStat
Namespace: (default namespace)
XML Schema: ns0.xsd

Example JSON

{ "id" : ..., "name" : "...", "total" : ..., "totalAlt" : ..., "extra" : "..." }

Stats call

This call returns general statistics about a series of downloads (i.e. dataset + any filters), it aggregates stats into the following categories;

Purpose ID Purpose
1 Personal interest
2 Educational purposes
3 Research and scientific analysis
4 Media publication
5 Commercial and consultancy work
6 Professional land management
7 Data provision and interpretation (commercial)
8 Data provision and interpretation (non-profit)
9 Statutory work

The id in this case matches one of the given purposes and the name is the label of that purpose

The total variable represents the total number of records downloaded for this purpose matching the given filters

The totalAlt variable represents the total number of downloads for this purpose matching the given filters

The extra variable will be blank

Example JSON

[{ "id" : 1, "name" : "Personal interest", "total" : 32, "totalAlt" : 2 }, { "id" : 4, "name" : "Media publication", "total" : 512, "totalAlt" : 4 }]

User Stats call

This call returns a set of stats for the top 5 users for downloads which match the given filters

The id in this case matches the userID of the user and the name is the name of that user

The total variable represents the total number of records downloaded by this user matching the given filters

The totalAlt variable represents the total number of downloads by this user matching the given filters

The extra variable is the email of that user

[{ "id" : 1, "name" : "Test User", "total" : 32, "totalAlt" : 1, "extra" : "test@email.com" }, ...]

Organisation Stats call

This call returns a set of stats for the top 5 organisations for downloads which match the given filters

The id in this case matches the organisationID of the organisation and the name is the name of that organisation

The total variable represents the total number of records downloaded by this organisation matching the given filters

The totalAlt variable represents the total number of downloads by this organisation matching the given filters

The extra variable will be blank

[{ "id" : 1, "name" : "Test Org", "total" : 32, "totalAlt" : 1, "extra" : "test@email.com" }, ...]

Combined Stats call

This call returns a set of above three calls in one JSON array, the first being a call to the stats resource, the second a call to the users resource and the third being a call to the organisations resource
